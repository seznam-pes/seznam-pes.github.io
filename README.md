# Kendrick Lamar Wiki

## 🚩 Cíl webu
### Web má posloužit všem fanouškům amerického interpreta Kendricka Lamara, aby zjistili, co třeba vydal za alba, jaké měl dětství, nebo jakých úspěchů za posledních pár let dosáhl.

<br>

## 👪 Cílová skupina
### Stránka míří převážně na fanoušky Kendricka Lamara, K. Dota, Kung Fu Kennyho,... prostě toho rappera z Comptonu co udělal skladbu Not Like Us. 

<br>

## 📜 Obsah
### Na webu se nachází hlavní stránka (homepage), pak takové ty povinné podstránky (cookies, GDPR, kontakt, zdroje), a pak ten hlavní obsah, který každého zajímá. Takže vydaná (i nevydaná) hudba, život, úspěchy a ocenění, a nějaké zajímavosti.
